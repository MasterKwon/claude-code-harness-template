## 기능 요구사항

### Screen (화면)
- [Must] 로그인 / 로그아웃 화면
- [Must] 회원가입 화면 (이메일, 비밀번호)
- [Should] 비밀번호 찾기 화면
- [Could] 소셜 로그인 (Google, Kakao)

### DB (데이터)
- [Must] 사용자 정보 저장 (이메일, 비밀번호 해시, 이름)
- [Must] 세션/토큰 관리
- [Should] 로그인 이력 기록

### API (서비스)
- [Must] 회원가입 API
- [Must] 로그인 / 로그아웃 API
- [Must] 내 정보 조회 API
- [Should] 비밀번호 변경 API

## 비기능 요구사항

| 항목 | 내용 |
|------|------|
| 성능 | API 응답 p95 < 300ms |
| 보안 | JWT 만료 7일, Refresh Token 사용 |
| 확장성 | MSA 구조로 서비스 독립 배포 가능 |
| 가용성 | Docker 기반 무중단 배포 |

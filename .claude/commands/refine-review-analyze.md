# 분석 리뷰 보완

기존 `docs/01.analyze/analyze-review-report.md`를 읽고 미진한 부분만 보완합니다.

## 실행 순서
1. `docs/01.analyze/analyze-review-report.md` 읽기 — 현재 리뷰 결과 확인
2. `docs/01.analyze/reviewed/` 전체 읽기 — 분석 문서 재확인
3. Read `.claude/skills/review-analyze.md` — 검토 항목 재확인
4. FAIL 판정 항목을 중심으로 미흡한 부분 재검토 및 보고서 보완

## 보완 후 확인
- [ ] 완전성 / 일관성 / 실현가능성 3개 축 모두 검토 완료
- [ ] FAIL 항목이 해소되었거나 보류 사유가 명확히 기록됨
- [ ] 분석 보완 사항이 있으면 `/refine-analyze-requirements` 또는 `/refine-analyze-asis` 실행 안내

**통과하면**: `docs/01.analyze/analyze-review-report.md`를 갱신하고 PASS 여부 재판정.
